# mi_nuevo_repo
Anuncio de datos de ventas de coches.
inicie creando un repositorio git con un archivo README.md y otro llamado .gitignore, luego me cree una cuenta Render, la vincule con GitHub. 
En visual Studio code cree mi espacio, en el instale los paquetes de panda, plotly-express y streamlit. Clore el repositorio de mi proyecto, agrege un directorio notebooks y dentro del directorio cree un jupyter llamdo EDA.ioynb, en el cual hice uso de la libreria plotly para contruir un histograma y por ultimo desarrolle un cuadro de mandos desde una aplicacion wed con un archivo llamado app.py, alli utilice la libreria streamlit para mostar un titulo, un cuadro con los anuncios de ventas de coche, un histograma y un boton para crear un histograma con los odometros.
